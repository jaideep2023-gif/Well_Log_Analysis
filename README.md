# Well_Log_Analysis
Well Log Analysis using Python 

## Overview
This project performs petrophysical analysis of LAS well log data using python.

## Features

-Triple Combo Plot
-Caliper vs Bit Size
-Neutron-Density Overlay
Total Porosity
Pickett plot
Velocity Analysis
Reflection Coefficient 
Compaction Trend

## Libraries

-Pandas
-Numpy
-Matploit
-LASIO

## Input

LAS files

## Output

-Triple Combo Logs
-Porosity Logs
-Pickett Plot
-Velocity Plots
